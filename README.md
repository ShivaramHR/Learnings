#Hangman (MIT 6.0001 – Problem Set 2)

This project implements a console-based Hangman game as part of MIT’s
6.0001 Introduction to Computer Science course.

## Concepts Practiced
- Control flow and loops
- State management
- Pure vs impure functions
- Input validation
- Spec-driven development

## Notes
This was one of my first multi-function programs (~220 lines) and
helped me understand how to structure larger programs.
